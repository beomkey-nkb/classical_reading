# 세종대 고전독서 관리 시스템

> 기획

2019. 03 ~ 2019. 04

> 개발 기간

2019. 04 ~ 2019. 06

> Description

웹페이지로만 운영되고 있는 고전독서 인증의 불편함을 개선하고자 어플리케이션으로 제작하게 되었다. 사용자의 편의성과 실용성에 중점을 두었다.

> 사용한 기술

로그인이 필요한 웹페이지를 scraping하기 위해 Phantomjs 사용, 자체 Node 서버와의 HTTP 통신, UserNotification 기능을 활용한 알림기능
